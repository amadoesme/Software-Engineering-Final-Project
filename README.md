# Book Library Web Application

A fully functional Django web application where users can browse books, search the catalog, leave comments and ratings, manage a shopping cart, favorite books, and more.
Built collaboratively as the Final Project for CS3337 – Software Engineering.

## Project Overview
   This application simulates a small online book library/store where users can:
  - View a catalog of books
  - Search by title, author, or ISBN
  - Rate and review books
  - Add books to a shopping cart
  - Mark books as favorites
  - Send messages through a contact form
  - View About Us information for the team
  - The system is fully styled using custom CSS and includes a modern, polished UI.

## Features Implemented
1. Search a Book
   - Search by title, author, or ISBN
   - Displays results in a responsive grid layout

2. Comments & Ratings
   - Users can leave comments
   - Rate books from 1–5

3. View average rating
   - Read previous reviews

4. Shopping Cart
   - Add books to cart
   - Increase or decrease quantities
   - Remove items
   - Clear entire cart

5. Favorites List
   - Save books using a heart icon
   - Remove favorites
   - View all favorites on a dedicated page
   - Includes animations and visual effects

6. Message Box
  - Allows users to send feedback messages
  - Simple and accessible contact form

7. About Us Page
  - Information about the team
  - Purpose and summary of the project
